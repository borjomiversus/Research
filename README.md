# Звіт: Вдосконалені структури даних на базі зв'язаних списків

## 1. Циклічний список (Circular Linked List)

### Що це таке?
Циклічний список — це різновид зв'язаного списку, у якому останній вузол не має вказівника на `null` (що зазвичай означає кінець списку). Натомість його вказівник `next` посилається назад на перший вузол (`head`), утворюючи замкнене коло. Циклічним може бути як однозв'язний (Singly), так і двозв'язний (Doubly) список.

### Особливості та реалізація
Як зазначається в навчальних матеріалах, важливою деталлю оптимізації циклічного списку є зберігання вказівника не на початок (`head`), а на кінець списку (`tail`).
* Оскільки `tail->next` завжди вказує на початок, ми маємо миттєвий доступ одночасно і до першого, і до останнього елементів.
* Це дозволяє здійснювати вставку елементів як на початок, так і в кінець списку за константний час $O(1)$ без необхідності обходити весь список.

### Практичне застосування
* **Round-Robin планування:** Алгоритми операційних систем часто використовують циклічні списки для диспетчеризації процесів. Процесору передаються задачі по черзі, і коли список доходить до кінця, він автоматично повертається до першого процесу.
* **Багатокористувацькі ігри:** Черга ходів гравців (після останнього гравця хід знову переходить до першого).
* **Відсутність null:** Під час обходу списку не потрібно робити перевірку на обрив, що спрощує логіку певних алгоритмів.

###  Занотовані питання під час дослідження
1. *Як запобігти нескінченному зацикленню під час обходу списку?*
   **Відповідь:** Замість стандартної перевірки `while (current != NULL)`, необхідно запам'ятати початковий вузол і використовувати умову `while (current != head)`.
2. *Чи існують ризики витоку пам'яті (memory leaks) при видаленні циклічного списку?* (Питання залишено для подальшого вивчення).

### 💻 Практична реалізація (С++)
```cpp
struct CNode {
    int data;
    CNode* next;
};

class CircularList {
    CNode* tail;
public:
    CircularList() : tail(nullptr) {}
    
    // Вставка в кінець за O(1)
    void push_back(int data) {
        CNode* newNode = new CNode{data, nullptr};
        if (!tail) {
            tail = newNode;
            tail->next = tail;
        } else {
            newNode->next = tail->next;
            tail->next = newNode;
            tail = newNode;
        }
    }
};
